iTips for Searching

For the most successful searches:

l Put quotation marks around searches with multiple words. For example, searching for "time off" finds only pages that include that exact combination of words. It does not find overtime, timesheet, etc.

l When you search for a single word, be as specific as possible. For example, timesheet would be better than time. Searching for time finds pages that contain time, overtime, timely, or timesheet.

l Use the Filter button in the Search box to filter the topics in which you search.

Adding Wired Live Sources for Video Conferencing

Each Prysm Application Suite deployment must be configured with different live sources based on the live source requirements from the Purchase Order. With wired live sources, participants at Prysm displays can share devices using a wired HDMI connection.

By default, four live sources are created for standard deployments, and one of these is an interactive source. For VC to be enabled, at least one of the live sources must be dedicated to the codec.

For details on how to identify the correct device ID, and map it to the correct connection port, refer to Section 9, Using the Device ID Mapper (DIM) Tool in the Prysm Platform Configuration Utility Guide, 520-00092-00.

To create or edit a wired live source:

  1. From the Deployments page, find the Deployment that you would like to configure live sources for and press Edit.
  2. In the drop-down menus, expand Menus >Live Sources.
  3. By default, four Live Sources are created for standard deployments, and one of these is an Interactive Live Source.

  4. Click Edit to edit an existing source or click Create Live Source to add a live source for the codec.

    The Live Source form is displayed.

  5. To create a Live Source for the Primary display output of the Codec, review the table below for the appropriate setting information, and then go to Step 7 and continue.

  6. Define the live source based on the settings in the table below (only settings that apply to VC are shown).

    Device Name Should only be populated to support legacy capture devices such as the BMD Intensity Pro. Should remain null for Magwell capture devices Text (leave blank unless using an older Prysm Application Appliance)
