Prysm Port Designations (LCD)

GPU Card, Port and Slot Definitions

GPU type and quantity may vary depending on the customer requirements for the Prysm solution being installed.

For All Cards:

  • SLOT ID – The physical slot PCI (Peripheral Component Interconnect) opening and PCI connector at the rear of the Application Appliance chassis are numbered from left to right beginning with 1, 2, 3, 4, 5,.. . Slot IDs are used to identify the location of various Application Appliance cards such as a Capture Card, GPU, or NIC card. For the X30, the range is 1 to 4.

For GPUs:

  • GPU ID – How the NVIDIA control panel identifies the card that a display is connected to (GPU 0, 1, 2). The GPU ID is dynamically assigned to a GPU when the card is connected to a display. In most cases the ID numbering starts with 0 and is sequenced from left to right. For the X30, possible values are 0.
  • GPU Logical Port Display ID – Is the logical connected display on any single GPU as viewed through the NVIDIA Control Panel. How the NVIDIA control panel identifies the port that a display is connected to (Logical Port ID 0, 1, 2, 3, 4...). The Logical Port ID is dynamically assigned to a port when it is connected to a display. The Logical Port ID numbering starts from bottom up from the first GPU to the next GPU card sequencing the "actual" display connections and begins with 0. It increments by 1 for each display connected to that single GPU (to a maximum of connected displays -1.)
  • GPU Physical Port ID – The Prysm defined GPU port identifier (Physical Port ID A, B, C, D). The Physical Port ID lettering starts with A and increments alphabetically from the bottom of the card to the top. For the K1200 it is A to D. (NVIDIA documentation refers to the physical ports numbered from 1 to 4, while the NVIDIA control panel refers to ports as 0 to 3.)
  • Broadcast Zone Port ID: The physical connector to which a broadcast zone output (e.g., to a codec) is made.
    • To run Broadcast, you must be running Prysm Application Suite 2.3.1 or later
    • For an Application Appliance with 1 GPU, the broadcast port is the top-most port
    • X30, K1200 – GPU Card 0, GPU PORT ID D (0.D)
    • X15, P600 - GPU Card 0, GPU PORT ID D (0.D)

For Capture Cards (CC):

  • CC Port ID: The physical connector across all capture cards (e.g. Magewell capture cards). Numbering starts with 1, on the left-most card in the system and increases from the top of the card to the bottom; where multiple cards are in a system, the CC Port ID is continuous across cards and continues to increment at the top of the next card to the immediate right.

  • Device ID: The logical designator of the capture source connected to a CC Port, and/or to a USB connector on the system. Numbering starts with 0 and increases to the maximum number (minus 1, because it is zero-ordered) of capture sources connected to the Application Appliance. There is an Audio and Video Device ID associated with every capture source. Device ID is used within the Touch Menu to bring up a live source. Device IDs are also associated with a USB-connected video camera and microphone input for use with Skype for Business.
