iTips for Searching

For the most successful searches:

l Put quotation marks around searches with multiple words. For example, searching for "time off" finds only pages that include that exact combination of words. It does not find overtime, timesheet, etc.

l When you search for a single word, be as specific as possible. For example, timesheet would be better than time. Searching for time finds pages that contain time, overtime, timely, or timesheet.

l Use the Filter button in the Search box to filter the topics in which you search.

Modifying Wireless Settings in the Touch Menu

With wireless live sources, participants at Prysm displays can share devices that support AirPlay services, including Mac computers, mobile phones, and tablets. This enables wireless device sharing.

To enable wireless device sharing for a Prysm Application Appliance, you must update the settings for that appliance.

  1. From the Deployments page, search for the settings profile that you want to enable wireless screen sharing on, and click Edit.
  2. Expand Customizations and click Top Level Menu.
  3. In the Live Sources (ShowSubMenu) section, click +to add a source.
  4. Complete the Menu Item form as follows.
    1. In the Name field, specify a name for the source. For example, Device List.
    2. In the Action field, specify AddDeviceList.
    3. Leave the Param field empty.
    4. Device List is added as a Live Source option.
  5. From the Deployments page, select your deployment and select Edit.
  6. Expand Enhancements and click WSS.
    For wireless screen sharing, define WSS settings as desired:

Streaming Buffer Size Seconds Number of seconds the live source consumer lags behind the live source producer. A higher number may result in smoother video playback, while a lower number is closer to real time. This setting affects the video stream for consumers but does not affect the producing location. Number (0-5) 1
Streaming Bitrate Mb/s The number of bits (in Mb) that can be transmitted along the network each second. This setting only affects the producing stream at the source location. Number (0.05 - 5) 1
Streaming FPS The number of frames per second streamed by the producer of the stream.

A higher value means more frames are streamed per second, so videos will stream more smoothly. A lower value means fewer frames are streamed per second, so video will be more choppy but a spreadsheet might be smoother. This setting affects the amount of network bandwidth required. Higher FPS settings consume higher bandwidth, and lower FPS settings consume lower bandwidth. 		Number (1-15) 10
Log Level

Sets the level of logging for WSS. From least to most logging, the options are:

Use Verbose only briefly for specific needs such as troubleshooting, because it creates large log files.
