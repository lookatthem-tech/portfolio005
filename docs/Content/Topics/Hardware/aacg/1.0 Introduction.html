This document does NOT cover field upgrade information for existing Application Appliances in the field. For field upgrade information refer to the appropriate Prysm Application Suite Upgrade Guide and field support bulletins available at https://prysm.kaybus.com/.

Support documentation described in Related Documents should be reviewed prior using this guide.

Audience

This Guide is intended for Prysm trained and qualified field engineers, partners and, and customer Network Administrators. This guide presumes that the audience has a full understanding of basic network provisioning and common Microsoft user authentication methods.

Safety Precautions

Only qualified service technicians should install and configure the Application Appliance. Basic electrical safety precautions should be followed.

Warnings

To ensure that the Application Appliance operates without error:

  • Do not install third-party software on a Prysm Application Appliance unless specifically required for new features or functionality as specified in a Purchase Order request.
  • Do not update Windows OS device drivers on the Prysm Application Appliance.
  • Do not enable Windows update or updates for any installed software on the Prysm Application Appliance.
  • Only use the applications and configuration files located in the C:\Tools directory in accordance with this and other Prysm documents.

Scope

This document covers the following tasks:
  1. Unpack and verify items (hardware/software) from the supplier/Prysm manufacturing.
  2. Connection of the Application Appliance.
  3. Configuration of the Application Appliance.
  4. Customization and Options.
  5. Confirmation of Installation and verification of Application Appliance functionality.
  6. Appendices (reference information).
