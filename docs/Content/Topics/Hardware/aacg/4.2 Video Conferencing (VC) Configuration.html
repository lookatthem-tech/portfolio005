iTips for Searching

For the most successful searches:

l Put quotation marks around searches with multiple words. For example, searching for "time off" finds only pages that include that exact combination of words. It does not find overtime, timesheet, etc.

l When you search for a single word, be as specific as possible. For example, timesheet would be better than time. Searching for time finds pages that contain time, overtime, timely, or timesheet.

l Use the Filter button in the Search box to filter the topics in which you search.

Configuring Video Conferencing (VC)

For Video Conferencing to work, a Prysm Application Appliance and Codec must be properly connected and the Video Conferencing (VC) feature must be enabled in the Web Admin. For connection information, see Video Conferencing (VC) Hardware Connections. For Web Admin information, see Enabling Video Conferencing in Web Admin.

Connecting and configuring a Prysm Application Appliance for Video Conferencing (VC) allows various functionality for the user. There are different scenarios that may be used with Video Conferencing.

User Places a Call to Another Resource

  1. A user walks into a room and selects Place a Call from the Welcome screen menu.
    1. The Prysm Go Experience screen with multiple source and sketchboard is displayed.
    2. The VC controls appear.
    3. The camera is set to preset 1.
  2. In the VC Call control area, the user selects the textbox and the on-screen keyboard appears.
  3. The user types the meeting URL into the textbox.
  4. The user selects Place a Call.
    1. If the call is answered, call participant video feed is displayed.

User Sends or Receives Content Between Resources

  1. A user is in a Codec based VC call.
  2. The user selects the sketchboard.
  3. The user toggles display content to enable the broadcast zone around the sketchboard content.
    1. Remote call participants can see the sketchboard on their displays.

  4. The user can switch between the sketchboard and live sources to update the broadcasted content.
