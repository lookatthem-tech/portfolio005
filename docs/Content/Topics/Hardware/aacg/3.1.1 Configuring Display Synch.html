iTips for Searching

For the most successful searches:

l Put quotation marks around searches with multiple words. For example, searching for "time off" finds only pages that include that exact combination of words. It does not find overtime, timesheet, etc.

l When you search for a single word, be as specific as possible. For example, timesheet would be better than time. Searching for time finds pages that contain time, overtime, timely, or timesheet.

l Use the Filter button in the Search box to filter the topics in which you search.

Configure Display Synchronization (Optional)

This feature is an added customized option, so if the Nvidia Genlock Quadro Sync card is part of the Application Appliance, configure the synchronization option as follows:

  1. Right-click anywhere on the Windows desktop and select Nvidia Control Panel.
  2. Navigate to Workstation and click on Synchronize Displays. (1)
  3. Under "1. The Timing Server Is . . ." select the radio button: On this system (2)
  4. Under "2. Select displays to lock to the server:" check the check box for Premium Mosaic. (3) This assumes that this step is performed AFTER Canvas Limiter has run.
  5. Click on the Server Settings button. (4)
  6. In the window that displays under "The synchronization pulses are based on:," select The server refresh rate (Internal timing) radio button for all installations EXCEPT for broadcast zone environments where a house genlock sync is present and has been connected via the BNC connector of the card. In that case, select the An external house sync signal radio button. Leave all other parameters at their defaults.
  7. Select Apply then OK to return to the Synchronize Displays window.
  8. Select Apply and then OK again to close the window.
  9. Verify that the house reference is active by doing one of the following:
    1. Verify that the House Sync LED light on the Nvidia Sync Card, at the rear of the Application Appliance, is blinking green
    2. Navigate to the Nvidia Control Panel > View System Topography and verify that the External sync signal is showing present
Note: These steps must be done each time an Application Appliance is rebooted and the house reference continues to be used.
