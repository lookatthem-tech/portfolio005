iTips for Searching

For the most successful searches:

l Put quotation marks around searches with multiple words. For example, searching for "time off" finds only pages that include that exact combination of words. It does not find overtime, timesheet, etc.

l When you search for a single word, be as specific as possible. For example, timesheet would be better than time. Searching for time finds pages that contain time, overtime, timely, or timesheet.

l Use the Filter button in the Search box to filter the topics in which you search.

Modifying Prysm Web Admin LCD Settings

For the Prysm 85 Dual Digital Workplace to render the images correctly on two displays, several settings in the Prysm Web Admin must be modified.

Note: Some of these settings may be included in the template applied when deploying a Prysm 85 Dual Digital Workplace.

Follow these steps to change the settings:

  1. Launch a web browser and sign into: https://admin.prysm.com/. (This only applies to signed in users. It does not apply to Prysm Go users.)
  2. Select Accounts and Impersonate the account for where the Application Appliance is being deployed.
  3. Navigate to Settings, locate the Settings Profile for the deployment you would like to edit, and select Edit.
  4. Expand Customizations and select Layout.
  5. Modify the following settings:
    1. Change the Modal Margin so that the third value listed is 3840.
    2. Change the File Menu Desired Height to 1280.
    3. Modify the Workspace Button Margin so that the third value listed is 3840 (this positions the Workspace Icon at the center of the left display LCD.)
    4. Select Save, after parameters are updated.
  6. Select Touch.
    1. Change Minimum Stage Item Width to 1.5.
    2. Make sure Touch Manipulation Threshold is set to 60.
    3. Press Save.
  7. Expand Experimental and select Browser.
    1. Change the Browser Default Width to 30.
    2. Change the Browser Max Width to 50.
    3. Change the Browser Default Height to 60.
    4. Press Save, after parameters are updated.
  8. Select Workspace Tray.
    1. Change Trailing Margin to a value of 50 so that the Workspace Tray is restrained (shows) only on the left LCD display.
    2. Select Save, after parameters are updated.
