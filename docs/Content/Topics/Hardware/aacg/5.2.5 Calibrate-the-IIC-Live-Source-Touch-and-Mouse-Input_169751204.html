iTips for Searching

For the most successful searches:

l Put quotation marks around searches with multiple words. For example, searching for "time off" finds only pages that include that exact combination of words. It does not find overtime, timesheet, etc.

l When you search for a single word, be as specific as possible. For example, timesheet would be better than time. Searching for time finds pages that contain time, overtime, timely, or timesheet.

l Use the Filter button in the Search box to filter the topics in which you search.

Calibrating the IIC Live Source Touch and Mouse Input

The touch and/or mouse input from the Prysm Application Workplace goes through the IIC to the Live Source computer. This requires calibration to synchronize the control.

  1. Before calibrating the IIC, verify that the Interactive Live Source has the following configuration:
    1. Live Source computer output resolution has an aspect ratio of 16:9 (for example: 1920×1080, 1280×720, etc.)
    2. Interactive Live Source does not supply a copyright-protected HDCP video signal output (do not use an Apple Mac laptop, iPad mini, etc., for the calibration)
    3. Live source is in projector only mode for IIC calibration, not extended desktop mode
    4. No icons or menu bars (remove, move, or hide) are in the upper-left corner of the desktop
  2. Open the VNC Viewer and enter the IIC (iPeps) IP address 192.168.1.101 into the Server field; then, click Connect.
  3. Enter the username: PrysmLPD and password: 500lumens
  4. In the VNC Viewer window, select Controls then Mouse Control.
  5. In the pop-up window, uncheck Absolute Mode and then click the Calibrate button and confirm by clicking OK. Do not move the mouse or touch the display during calibration.
  6. If the calibration is successful, a dialog box appears stating the unit is operating in Windows Enhanced Mk1 mode. Any other mode indicates that the calibration was not successful.
  7. In such a case, confirm that both the video output and a USB connection to the Application Appliance are correct. Then, repeat the calibration steps until the success message Windows Enhanced Mk1 mode appears.
  8. Re-engage (check) Absolute Mode to lock the calibration and close the mouse calibration window.
  9. Move the mouse within the VNC Viewer to verify accurate mouse/cursor placement relative to the remote host. There should be little to no delay or shadow as you move the mouse across the Prysm Application Workplace. Note that a small degree of latency between the two cursors is normal when moving the mouse.
  10. Once successfully calibrated and tested, label the iPeps component of the IIC with the assigned IP address, and update the Application Appliance Installation Record.
