iTips for Searching

For the most successful searches:

l Put quotation marks around searches with multiple words. For example, searching for "time off" finds only pages that include that exact combination of words. It does not find overtime, timesheet, etc.

l When you search for a single word, be as specific as possible. For example, timesheet would be better than time. Searching for time finds pages that contain time, overtime, timely, or timesheet.

l Use the Filter button in the Search box to filter the topics in which you search.

Verifying IIC to Application Appliance Connectivity

The Prysm Application Suite relies on both a network and a video connection to a Live Source. Complete the following steps to verify the IIC is correctly connected to the Application Appliance. If there are multiple IICs, disconnect all iPEPS devices except the one to be verified.

  1. Verify Network Connectivity.
    1. Open a command prompt
    2. Issue the ‘ping’ command at the IIC’s IP Address
    3. Verify a reply is received from the device
  2. Verify Video Connection – Ensure the Live Source is connected to a capture card input and outputting video.
    1. Using VLC media player, navigate to Media > Open Capture Device.
    2. Select the Capture Device tab and the Video Device Name drop down until the video is displayed by the Live Source.
  3. Verify possible Cross-link between Video and Network Connections.
    Cross-link of video and network connections within the same rack tray may occur in installations with multiple IICs (i.e., a Live Source may be connected to a different IIC input than the target Adderlink iPEPS unit due to incorrect IP Address mapping).
    1. Connect a Live Source directly to the Gefen HDMI Detective Plus.
    2. From the Application Appliance, open a VNC Viewer and enter the IP address of the target iPEPS.
    3. Verify that the test video from the Live Source appears in the VNC window.
    4. Once this video has been verified, the IP Address of the iPEPS can be reliably mapped to the capture card input DeviceIndex in the Web Admin.

  4. Navigate to Web Admin > Deployments.

    1. Press Edit on the desired Setting Profile to make changes.
    2. The Edit Deployment screen displays.
    3. Expand Customizations then select Top Level Menu.
    4. The Edit Top Level Menu Settings view displays.
    5. Select Sub-Level Menu item to be verified and click Edit to show IP information.

    6. Note the IP address and VNC Capture Interactive Live Source Device Index as shown in the above example.
