iTips for Searching

For the most successful searches:

l Put quotation marks around searches with multiple words. For example, searching for "time off" finds only pages that include that exact combination of words. It does not find overtime, timesheet, etc.

l When you search for a single word, be as specific as possible. For example, timesheet would be better than time. Searching for time finds pages that contain time, overtime, timely, or timesheet.

l Use the Filter button in the Search box to filter the topics in which you search.

Configuring the LCD Settings

The LCD Config Utility provides a simple GUI to configure Mosaic and Warp using algorithms based on best practices. Entering settings in the LCD Config Utility provides a preview of the final location and coordinates of the final mosaic video image.

Entering LCD Array Settings

  1. Select LCD Config from the Prysm Platform Configuration Utility left side menu.
  2. Use the available settings to configure the LCD array:

Saving LCD Array Settings

  1. After the correct settings have been applied, select Save.
  2. You will be prompted to enter a name for this profile... for the configuration file.
  3. Select Save after entering a name.
