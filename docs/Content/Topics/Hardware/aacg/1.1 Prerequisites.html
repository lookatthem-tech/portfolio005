iTips for Searching

For the most successful searches:

l Put quotation marks around searches with multiple words. For example, searching for "time off" finds only pages that include that exact combination of words. It does not find overtime, timesheet, etc.

l When you search for a single word, be as specific as possible. For example, timesheet would be better than time. Searching for time finds pages that contain time, overtime, timely, or timesheet.

l Use the Filter button in the Search box to filter the topics in which you search.

Prerequisites and Installation Guide Flowcharts

The Prysm Application Appliance and Prysm Application Suite software are configured to default factory settings. Additional field configurations are necessary to reflect the integration and functional requirements for each customer. Before configuring the Application Appliance, review this section and gather all necessary materials.

Prerequisites

Field installation and support personnel must be familiar with the following applications and procedures:

  • Prysm Digital Workplace and touch sensor installation and operation
  • Microsoft Windows 8.1 and Windows 10 desktops, applications, directory structure, navigation, text editing through Notepad
  • Nvidia control panel for navigating display resolutions settings
  • Basic Prysm Application Suite end user functions
  • Salesforce access for viewing and editing the Application Appliance Installation Record
  • VLC, VNC, and Windows network configuration process
  • VC or Video Conferencing Integration
  • Third-Party Applications Integration
